# Keyboard Shortcuts - Aether

Aether is designed to be keyboard-first for maximum productivity. Every action has a keyboard shortcut.

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘K` (Mac) / `Ctrl+K` (Win) | Open Command Palette | Universal command center - search everything |
| `⌘N` (Mac) / `Ctrl+N` (Win) | New Conversation | Start a new chat |
| `⌘B` (Mac) / `Ctrl+B` (Win) | Toggle Sidebar | Show/hide conversations sidebar |
| `⌘,` (Mac) / `Ctrl+,` (Win) | Open Settings | Access app settings |
| `⌘⇧T` (Mac) / `Ctrl+Shift+T` (Win) | Toggle Theme | Switch between light/dark mode |
| `Esc` | Close Modal/Palette | Close any open modal or command palette |

## Command Palette (⌘K)

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate commands |
| `Enter` | Execute selected command |
| `Esc` | Close palette |
| Type to search | Fuzzy search all commands |

### Quick Commands (via Command Palette)
- "New Conversation" - Start new chat
- "Google Cloud Settings" - Configure GCP
- "API Keys" - Manage API keys
- "Switch to [Model]" - Change AI model
- "Export" - Export conversation
- "Delete" - Delete current conversation

## Composer (Message Input)

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line (without sending) |
| `/` | Activate slash commands |
| `Esc` | Clear input |

### Slash Commands
- `/model` - Switch AI model
- `/export` - Export conversation
- `/persona` - Change persona
- `/clear` - Clear input

## Sidebar Navigation

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate conversations |
| `Enter` | Open selected conversation |
| Click conversation | Open that conversation |
| Hover + Delete icon | Delete conversation |
| Hover + Pin icon | Pin/unpin conversation |

## Settings (⌘,)

| Shortcut | Action |
|----------|--------|
| `Esc` | Close settings |
| `Tab` | Navigate between tabs |
| Click tab | Switch to that settings tab |

### Settings Tabs
- **Google Cloud** - GCP credentials and region
- **API Keys** - OpenAI API key
- **Personas** - Custom AI personalities
- **Usage** - Track model usage
- **Data** - Export/import data

## Tips for Power Users

1. **Command Palette is Your Friend** - Press `⌘K` to access everything instantly
2. **Keyboard Navigation** - Navigate entire app without touching the mouse
3. **Slash Commands** - Type `/` in composer for quick actions
4. **Quick Filters** - Use sidebar filters (All, Today, Pinned) for organization
5. **Theme Toggle** - Quickly switch themes with `⌘⇧T`

## Accessibility

- **Screen Readers** - Full VoiceOver/NVDA support
- **High Contrast** - Works with system high contrast mode
- **Keyboard Only** - Every feature accessible via keyboard
- **Focus Visible** - Clear focus indicators on all interactive elements
- **Reduced Motion** - Respects `prefers-reduced-motion` system setting

## Platform-Specific Notes

### macOS
- Use `⌘` (Command) key for shortcuts
- Global menu integration
- Native keyboard shortcuts work (⌘Q to quit, ⌘M to minimize)

### Windows
- Use `Ctrl` key for shortcuts
- Alt+F4 to close window
- Windows key combinations work as expected

### Linux
- Use `Ctrl` key for shortcuts
- Standard Linux desktop shortcuts supported
